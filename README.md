# Claude skills

## /grill

Inspired by Matt Pocock [`/grill-me`](https://www.aihero.dev/skills-grill-me) skill, and adapted to the specs workflow.

It can be used with an idea (just type something) or a spec file.
It can also generate a spec file from an idea or a plan from a spec file.

**Install skill with**:

```sh
npx skills add pierre-lecocq/claude-skills --skill=grill -y
```
